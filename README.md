## Prerequisites

- [k6](https://k6.io/docs/getting-started/installation)
- [NodeJS](https://nodejs.org/en/download/)
- [Yarn](https://yarnpkg.com/getting-started/install) (optional)

## Installation
* **Install dependencies**

```bash
$ yarn install
```

## Running the test

```bash
$ yarn webpack
```
```bash
$ k6 run dist/get-200-status-test.js
```
